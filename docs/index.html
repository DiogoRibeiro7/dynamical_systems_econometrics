Research repository

Dynamical Systems, Rare Events, and Econometrics

A reproducible Python project studying crises and tail dependence through dynamical-systems-inspired extreme value theory, recurrence diagnostics, and macro-financial data workflows.

Core questions

Can crisis regimes be treated as rare events with dependence rather than isolated iid shocks?

How much clustering is present in exceedances, and what does that imply for return times and stress persistence?

What is implemented

Simulation utilities, extremal-index estimators, threshold diagnostics, return-time tools, empirical loaders, notebooks, figure scripts, and CI-backed quality checks.

Quick start

pip install -e ".[dev]"

python scripts/run_smoke_experiment.py

python scripts/generate_article_figures.py

Data contract

Every loader is normalized to date, series_id, value so diagnostics remain source-agnostic.

Reproducibility

The repository ships with tests, Ruff, MyPy, release tagging, citation metadata, and GitHub Actions workflows.
