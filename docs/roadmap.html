Roadmap

This roadmap distinguishes what the repository already supports well from what is planned and what should remain outside scope unless the evidence and maintenance story become stronger.

Already solid

  • core package layout with reusable simulation, extremes, recurrence, and plotting modules
  • CLI and script entry points for synthetic workflows and article-facing figure generation
  • public documentation covering setup, commands, validation, results, and contribution rules
  • tests for contracts, edge cases, pipeline wrappers, and figure output behavior

Planned extensions

  • richer empirical data adapters with clearer provenance metadata
  • more comparative diagnostics between recurrence methods and baseline econometric summaries
  • additional article-quality figures that emphasize uncertainty and threshold sensitivity
  • stronger publication-facing packaging around claims traceability

Needs caution before expansion

  • automated empirical narratives that read outputs as conclusions
  • large new model families without tests, contracts, and interpretation guidance
  • workflow growth that increases complexity faster than reproducibility coverage

Explicit non-goals for now

  • pretending fallback synthetic runs are substitutes for empirical validation
  • turning the repository into a general macroeconomic forecasting platform
  • adding dashboards or interfaces that outrun the underlying evidence quality
