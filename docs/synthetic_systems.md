# Synthetic Systems

## Purpose

The synthetic systems in this repository are benchmark processes for validating rare-event and recurrence diagnostics before applying them to empirical macro-financial data.

## Implemented systems

- logistic map
- AR(1)
- GARCH(1,1)
- intermittent benchmark map
- coupled logistic maps

## Interpretation guardrail

None of these systems should be interpreted as realistic economies. They are controlled examples with different recurrence and clustering properties.
