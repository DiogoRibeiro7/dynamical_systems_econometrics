# Econometric Baselines

## What this adds beyond standard econometrics

Standard econometric diagnostics such as autocorrelation, rolling volatility, and drawdowns describe persistence, scale, and downside behavior. The rare-event and recurrence layers add explicit focus on threshold crossings, clustering, and return times between stress episodes.

## Implemented baselines

- autocorrelation summaries
- rolling volatility
- drawdown paths
- regime summaries
- alignment with event diagnostics

## Caution

These baselines and the rare-event diagnostics are complementary descriptive tools. They are not causal estimators.
