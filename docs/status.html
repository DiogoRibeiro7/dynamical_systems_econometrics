Project status

This page summarizes the current implementation state of the repository as a research package. It is intended as a quick orientation layer between the landing page and the detailed audit/review documents.

Implemented

  • contracts and config helpers
  • simulation benchmarks for univariate and multivariate stress examples
  • threshold exceedances, runs declustering, and extremal-index diagnostics
  • return-time, recurrence, and survival summaries
  • multivariate joint-stress and stress-state diagnostics
  • offline-first data loading and catalog validation
  • lightweight econometric baselines and alignment helpers
  • experiment orchestration with saved tables, figures, and JSON summaries
  • article figure generation with matching source CSV files
  • public docs site covering commands, configs, data, methods, notebooks, figures, and troubleshooting

Partially implemented

  • empirical case-study workflows depend on user-supplied public data in canonical format
  • notebook execution is documented and structured, but not automated in tests
  • the original and prompt-aligned notebook tracks coexist, which is flexible but not fully simplified

Not implemented or intentionally out of scope

  • causal identification layer
  • strong structural estimation of a macroeconomic dynamical system
  • claims that observed economies are literal deterministic maps
  • exact crisis-date forecasting from return-time diagnostics
