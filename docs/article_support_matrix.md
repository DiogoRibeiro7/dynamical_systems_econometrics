# Article Support Matrix

## Supported by current code and saved outputs

- synthetic comparisons of clustered versus more isolated extremes
- threshold exceedance extraction
- runs-based extremal-index summaries
- return-time and recurrence summaries
- multivariate joint-stress timelines
- descriptive comparison with baseline volatility-style diagnostics
- reproducible article figures with matching source CSV files

## Partially supported

- empirical case studies, if a real public dataset is supplied locally in canonical format
- regime comparisons, if the researcher defines regimes explicitly and saves them in outputs

## Not supported by current code

- causal identification
- policy attribution
- structural estimation of a macroeconomic dynamical system
- deterministic-law claims about observed economies
- exact crisis forecasting

## Use in article drafting

When writing the article, map each section to one of the categories above before finalizing the prose. If a claim falls into the unsupported category, it should either be removed or reframed.
