Method notes

The repository is motivated by rare-event ideas from the statistical study of dynamical systems, but it applies those ideas to observed macro-financial time series as descriptive diagnostics rather than as proof that economies are literal deterministic maps.

Conceptual translation

Dynamical-systems concept Econometric analogue
State spaceLatent macro-financial state of the economy
OrbitObserved time path of the economy
ObservableInflation, unemployment, spreads, returns, volatility, output
Rare setCrisis region or stress threshold
Hitting timeTime until the system enters a crisis region
Return timeTime between visits to comparable stress states
Extremal indexDegree of clustering in extreme events
Compound Poisson limit intuitionCrises may arrive in bursts rather than as isolated shocks
Multivariate extremesJoint stress across several economic variables

Why this matters

Standard econometric tools describe dependence through autocorrelation, conditional heteroskedasticity, and regime-like behavior. The rare-event layer adds a different question: whether threshold exceedances arrive independently or in clusters, and how quickly the system returns to dangerous regions after leaving them.

Repository workflow in method terms

  • Construct benchmark systems with different recurrence structures.
  • Define stress regions using thresholds or transformed observables.
  • Measure exceedances, clustering, and recurrence.
  • Extend the same logic to joint stress across several variables.
  • Compare the descriptive outputs with baseline econometric summaries.

Caution

The interesting claim is not that the economy is a simple chaotic map. The interesting claim is that observed economic series can exhibit recurrence, clustering, and joint-tail structure that are naturally described using tools developed in the dynamical-systems rare-event literature.
