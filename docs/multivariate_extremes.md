# Multivariate Extremes

## Purpose

Multivariate stress diagnostics capture dates on which more than one variable is simultaneously in a high-stress region.

## Implemented tools

- long-to-wide panel conversion
- joint exceedance tables
- empirical tail-dependence probabilities
- weighted stress-state indices

## Guardrail

Joint stress is a diagnostic description of synchronized extreme behavior. It is not proof of structural or causal linkage.
