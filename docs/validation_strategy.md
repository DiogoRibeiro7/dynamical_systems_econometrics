# Validation Strategy

## What is validated directly

- long-format and event-frame schema contracts
- parameter bounds for simulations
- hand-crafted extremal and recurrence sequences
- grouped transforms
- multivariate stress summaries
- experiment-output persistence

## What is sanity-checked rather than proven

- qualitative clustering differences between benchmark processes
- plotting behavior through smoke-level path creation

## What tests cannot prove

- asymptotic extreme-value validity in short empirical samples
- substantive empirical claims without explicit data runs
- causal mechanisms
