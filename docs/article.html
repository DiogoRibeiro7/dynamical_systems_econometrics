Article guide

This page connects the article-facing layers of the repository: the draft narrative, the stable figures, the source CSVs behind those figures, and the claim guardrails that keep the public prose aligned with what the code actually supports.

Article-facing repository assets

  • Article draft
  • Figure gallery
  • Claims checklist
  • Article support matrix

What should drive the prose

  • saved figures generated by code
  • matching source CSV files for each article-facing figure
  • saved experiment tables and summaries where empirical claims are made
  • guardrail documents that separate descriptive structure from causal claims

What should not drive the prose

  • synthetic fallback outputs presented as empirical findings
  • notebook-only intermediate state that was never saved
  • causal language unsupported by a separate identification design
  • deterministic-law claims about observed economies

Recommended writing flow

  1. Start from the project status page to see what is implemented.
  2. Use the figure gallery to identify stable article-facing visuals.
  3. Check the support matrix before turning a result into prose.
  4. Use the claims checklist as a final pass before publishing language.

Interpretation rule

The interesting empirical claim is that observed economic series can exhibit recurrence, clustering, and joint stress structure that these diagnostics describe. The repository does not support stronger claims about literal deterministic dynamics or causal mechanism by itself.
