Results guide

This page separates generated artifacts from substantive claims. The repository produces reproducible outputs, but those outputs still need interpretation discipline before they are treated as evidence about crisis dynamics or macro-financial structure.

Main output families

  • synthetic experiment summaries for controlled behavior checks
  • empirical pipeline outputs when external public data is configured
  • article-facing figures and matching source CSV files
  • diagnostic plots for thresholds, exceedances, recurrence, and clustering

Operational commands

python scripts/run_smoke_experiment.py
python scripts/run_empirical_experiment.py
python scripts/generate_article_figures.py

What counts as a stable artifact

  • version-controlled source code, configuration, and article-facing figure scripts
  • named PNG and CSV bundles intended for article drafting and inspection
  • tables and summaries generated from deterministic synthetic seeds where configured

What should be treated as exploratory

  • single-threshold empirical findings without stability checks
  • short-sample tail estimates presented without uncertainty context
  • plots produced from fallback synthetic data and then described as real-world evidence

Reading results correctly

A figure or table is not a claim by itself. The stronger the inference, the more it depends on sample quality, threshold robustness, estimator sensitivity, and whether the repository's supported-claim guardrails actually cover that interpretation.
