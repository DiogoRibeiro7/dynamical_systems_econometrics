# Article Figures

## Scope

Figures in this directory are generated by repository scripts rather than drawn manually.

- `simulated_orbit_and_observable.png`
- `threshold_exceedances.png`
- `clustered_vs_isolated_extremes.png`
- `return_time_distribution.png`
- `extremal_index_by_threshold.png`
- `macro_financial_crisis_timeline.png`

## Source CSV assets

- `simulated_orbit_and_observable_source.csv`
- `threshold_exceedances_source.csv`
- `clustered_vs_isolated_extremes_source.csv`
- `return_time_distribution_source.csv`
- `extremal_index_by_threshold_source.csv`
- `macro_financial_crisis_timeline_source.csv`
- `multivariate_stress_heatmap_source.csv`

Each figure now has a matching CSV source so article claims can be traced to saved structured outputs rather than unsaved plotting state.

## What these figures support

- conceptual comparison between clustered and isolated extremes
- descriptive recurrence and return-time summaries
- descriptive multivariate stress timelines

## What they do not support

- causal claims
- literal deterministic-law claims about observed economies
